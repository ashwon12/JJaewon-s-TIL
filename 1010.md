### 파이썬의 클래스란?

객체 (object)는 데이터와 이를 처리하기 위한 메소드의 묶음을 일컫는 말이다.

특정한 종류의 객체를 만드는 형틀을 Class라고 하고 클래스부터 만들어지는 객체를 그 클래스의 Instance 라고 한다.

파이썬에서는 모든 것이 객체로 구현된다.

### 클래스 정의하기

```python
class Class_NAME :
	def __init__(self,initial) :
			멤버 초기화
	def method ():
			메소드 정의
```

- 클래스 내에 정의된 함수들 = method

    메서드의 첫 번째 인자는 `self` 여야 한다. 

    ⇒ 파이썬은 메서드의 첫번째 인자로 항상 인스턴스가 전달된다.  ***인자를 전달하지 않아도 메서드를 호출하면 인스턴스가 전달이 된다.***

```python
class BankAccount :

		#초기자 객체를 생성할 때 넘긴 인자들을 받아와 주는 곳
    def __init__(self,balance,name,number):
        self.balance = balance
        self.name = name
        self.number = number

    #출금메소드
    def withdraw(self,money) :
        self.balance = self.balance-money
        return self.balance

    #입금메소드
    def deposit(self,money):
        self.balance = self.balance+money
        return self.balance

a1=BankAccount(10000,"홍길동","123456") #은행계좌의 객체 a1을 만들어줌
money=a1.deposit(30000) # 클래스의 메서드를 호출시킴

print(money)
```
