## 안드로이드

### 단방향성 intent 전달

- **MainActivity → 2Activity 발신**

```java
Intent intent = new Intent(MainActivity.this, 2Activity.class);
intent.putExtra("num1",num1);
intent.putExtra("num2",num2);
startActivity(intent);
```

- **2Activity에서 intent 수신**

```java
/* 값 받아오기 */
Intent rxIntent = getIntent();
Bundle extras = rxIntent.getExtras();

if(extras !=null){
    num1 = extras.getInt("num1");
    num2 = extras.getInt("num2");
}
```

### 양방향성 intent 전달

- **MainActivity → 2Activity 발신**

```java
Intent intent = new Intent(MainActivity.this, 2Activity.class);
intent.putExtra("num1",num1);
intent.putExtra("num2",num2);
startActivityForResult(intent,0); // 0은 requestCode , 사용자 지정
```

startActivityForResult() 메소드를 사용하여 응답받은 intent를 처리한다.

첫번째 파라메터는 보내는 intent 값이고 두번째 파라메터는 requestCode로 사용자 지정 숫자이다.

2Activity에서 수신은 단방향일 때랑 똑같이 사용하면 된다.

- **2Activity → MainActivity 발신**

```java
Intent outIntent = new Intent(getApplicationContext(),MainActivity.class);
outIntent.putExtra("resultVal",(num1+num2));
setResult(RESULT_OK,outIntent);
finish();
```

- **MainActivity 수신**

```java
@Override
    protected void onActivityResult(int requestCode, int resultCode, @Nullable Intent data) {
        super.onActivityResult(requestCode, resultCode, data);
        if (resultCode == RESULT_OK) {
            int resultValue = data.getIntExtra("resultVal", 0);
            result.setText(String.valueOf(resultValue));
        }
    }
```

requestCode가 RESULT_OK( 2Activity에서 보냄) 이고 getIntExtra에서 requestcode가 0일 경우 값을 받아온다.

## 라디오버튼 사용법 정리

RadioGroup 안에 RadioButton들이 들어있다.

```java
rg=(RadioGroup)findViewById(R.id.rg); // xml에서 radiogroup과 연결

int id = rg.getCheckedRadioButtonId(); 
if(id == R.id.rb1) {
      Log.d("선택된 값 ", id.getText().toString()); 
}else{
    Toast.makeText(MainActivity.this,"버튼을 선택해주세요",Toast.LENGTH_SHORT).show();
}
```

id에 들어오는 값은 rg 라디오 그룹에 속해있는 라디오버튼 중 선택된 버튼의 id값이 들어온다!
