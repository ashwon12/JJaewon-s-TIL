셀레니움은 웹 스크래핑을 할 때 특정 행동이 취해졌을 때 일어나는 값들에 대해서 값을 가져오는 것이다. 크롬 **브라우저를 조작**해서 원하는 값들을 가져올 수 있다.

### 셀레니움이랑 Beautifulsoup 같이 사용하기

- **패키지 설치**

    `pip install selenium`

- **chrome driver 설치하기**
- **chrome 열기**

```python
from selenium import webdriver
from bs4 import BeautifulSoup

path = "C:/Users/rhdwn/Documents/chromedriver_win32/chromedriver"
driver = webdriver.Chrome(path)
```

path는 chrom driver가 저장되어 있는 위치를 입력해준다.

```python
key = key
url = 'http://q-net.or.kr/crf005.do?id'+key+'&gSite=Q&gId=&div=1&obligFldCd='
driver.get(url)
req = driver.page_source
soup = BeautifulSoup(req, 'html.parser')
wrap = soup.select("body > ul > li")

# 브라우저 닫기
driver.close()
```

특정 사이트에서 반복적으로 바뀌는 key값을 모아서 반복문을 통해 url을 변경해주고 크롬으로 창을 열어준다. 그런 다음 그 페이지에서 값을 크롤링 해오면 된다.
