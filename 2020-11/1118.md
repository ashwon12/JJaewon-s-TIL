## ajax로 페이지 이동하면서 데이터 전달하기

#### 첫번째 페이지에서 입력받은 값을 url로 전달

```jsx
  //window.location.href = /이동하려는 html의 url?변수=전달하려는값
  window.location.href ='/listPage?data='+data
```


#### 해당 url에서 값을 받아서 html로 데이터 전달

```python
  @app.route('/listPage')
  def listPage():
    major = request.args.get('data')
    return render_template('listPage.html', data=data)

```

#### 두번째 페이지에서 데이터 받아서 사용하기

```html
    <meta id="major" content="{{}}">
    <div class="text_container">
```
 {{data}} 이 문법은 html 코드를 직접적으로 입력할 때 사용하는 코드이다.
 그렇기 때문에 html에서만 사용 가능하다.
 
  meta 태그로 우선 html에 뿌려준다.

<br>

