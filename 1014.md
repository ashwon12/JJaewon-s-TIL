## Ajax

자바스크립트로 네트워크 요청을 하는 통신 방법,                               **비동기** 서버 통신   방식, 제이쿼리에서 부르는 방법

- 클라이언트 → 서버 **GET** 요청

    웹은 HTTP라는 통신 규약을 따르는데 HTTP에는 다양한 요쳥 메소드가 있다. 정보를 요청하는 과정에서는 대부분 ***GET방식을 사용***하고 있다.

    ⇒ [다양한 HTTP 요청 메서드](https://developer.mozilla.org/ko/docs/Web/HTTP/Methods)

- **HTTP의 GET 방식**

    `https://movie.naver.com/movie/bi/mi/basic.nhn?code=161967`

    코드에서 ? 기준으로 앞부분이 <서버 주소 >, 뒷부분이 <영화 번호 > 이다.

    서버가 클라이언트로 넘겨주는 데이터의 표현방법이 `JSON` 이다. 

    - JSON은 자바스크립트에서 `오브젝트를 표현하는 방식`이다.
    - Key : Value 로 이루어져 있어 자료형 Dictionary와 유사하다.

- **사용예제**

```jsx
$.ajax({
  type: "GET", // GET 방식으로 요청한다.
  url: "http://openapi.seoul.go.kr:8088/6d4d776b466c656533356a4b4b5872/json/RealtimeCityAir/1/99",
  data: {}, // 요청하면서 함께 줄 데이터 (GET 요청시엔 비워두세요)
  success: function(response){ // 서버에서 준 결과를 response라는 변수에 담음
    console.log(response) // 서버에서 준 결과를 이용해서 나머지 코드를 작성
  }
})
```
