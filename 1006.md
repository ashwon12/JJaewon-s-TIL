#### css로 가운데 정렬 시키는 방법 = `display:felx` 사용하기

felx 레이아웃의 속성들을 사용해서 레이아웃을 수정할 수 있음

```css
.login-title {
    color: white;
    display: flex;
    flex-direction: column; /* 세로로 정렬 하겠다는 뜻*/
    justify-content: center; /*flex-direction에 대한 정렬*/
    align-items: center; /*justify-content의 수직방향을 정렬*/
}
```

속성들을 사용하면 엄청나게 다양한 레이아웃을 구성할 수 있는데 조금 더 공부해야 할 듯!!!
