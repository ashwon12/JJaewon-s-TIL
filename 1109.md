## aws 
내 컴퓨터에서 서버를 실행시키기 위해 24시간 컴퓨터를 돌릴 수 없으니까

클라우드에서 서버를 구매해서 공개 IP주소를 얻어 나의 프로젝트와 연결시켜야 한다.

aws는 클라우드 서비스 중 하나이다. 

### 클라우드 사용과정

1. 컴퓨터-> EC2 -> 인스턴스로 들어간다.

2. 원하는 운영체제가 탑재된 서버를 선택한다.

3. key를 받는다.( pem 파일) key는 단 한번만 다운받을 수 있고 절대 남에게 공개해서는 안된다.

4. AWS의 보안그룹에 내가 사용할 서버들을 등록해준다.

5. ubuntu에서 아래의 명령어를 입력해서 구매한 서버의 컴퓨터로 접속한다.

    ```sss -i key pair ubuntu@구매한 서버의 ip```


    종료할 때는 exit 명령어를 입력해서 내 컴퓨터로 돌아온다.

6. FIle Zilla 는 파일을 내 컴퓨터에서 서버 컴퓨터로 전송해주는 프로그램이다.
  file zilla와 구매한 서버의 ip를 연동시키고 파일을 보내준다.
  
7. ubuntu를 통해 전달받은 파일의 상태를 확인하고 실행시킬 수 있다.
