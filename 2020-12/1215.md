### mongodb update 연산자

- `$push` : 배열에 값 추가

만약에 [1,2]라는 배열에 [3,4,5]를 넣고 싶은에 push만 사용하면 [1,2,[3,4,5]]가 된다. 3,4,5를 꺼내서 넣고 싶으면 `$each`를 사용한다.

```python
  { $addToSet: { 필드: { $each: 배열 } } }
```

- `$set` : 배열 업데이트 ( 값을 아예 리셋한다음 추가)

```python
  db.users.update_one({'name': '덤블도어'}, {'$set': {'age': 19}})
```

- `$or` : or 연산자
```python
    db.users.find({
      $or: [
      {"name": "덤블도어"}, 
      {"age":19}
      ]
    })
```
