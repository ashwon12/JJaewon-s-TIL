## 오픈소스

### 공개SW 라이센스

- **지식 재산권**

    지적 능력을 가지고 만들어 낸 창작물에 대한 권리

    1. 저작권( 미술,음악,영화,소설,소프트웨어) : 저작물을 창작한 때부터 발생하여 보호, 
    2. 산업재산권(특허권,상표권,디자인,영업비밀) : 특허청에 등록받아야 함

- 소프트웨어의 지식 재산권
    1. 순서도 작성(알고리즘) → 특허출원의 경우 , 특허권으로 보호
    2. 프로그래밍(소스코드) → 저작권에 의해 보호 
    3. 실행코드 생성(실행코드) → 저작권에 의해 보호 

- 라이선스 (license)

    재산권을 가진 자가 타인에게 대가를 받고 그 재산권을 사용할 수 있도록 부여하는 계약 

    Liscensor : 재산권을 가진 자

    Licensee : 재산권을 대여 받는 자

## python

### 예외처리

- 문법

    ```python
    try :
    	#실행하는 명령들
    except 예외명 as 변수 :
    	#예외가 발생할 때 수행할 명령들
    else :
    	#예외가 발생하지 않을 때 수행할 명령들
    ```

- 대표적인 예외 종류

  `NameError` 지역 이름이나 전역 이름이 발견되지 않음
  
  `ValueError` 타입은 맞지만 값의 형식이 잘못됨
  
  `ZeroDivisionError` : 0으로 나눔
  
  `IndexError` : 첨자(index)가 범위를 벗어남
  
  `TypeError` : 타입이 맞지 않음
